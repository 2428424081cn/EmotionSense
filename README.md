# EmotionSense

实时面部表情识别系统，基于 TensorFlow.js 和 face-api.js 构建，支持 7 种基本表情的实时检测与可视化。

## 功能特性

- 实时摄像头面部表情识别
- 7 种表情检测：开心、悲伤、愤怒、惊讶、恐惧、厌恶、中性
- 置信度实时显示
- 历史记录保存
- 统计面板

## 使用方法

1. 克隆仓库
2. 使用本地服务器运行（如 `npx serve` 或 VS Code Live Server）
3. 允许浏览器访问摄像头

## 技术栈

- TensorFlow.js
- face-api.js
- 原生 JavaScript
